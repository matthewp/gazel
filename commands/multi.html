multi

Begins a multi-command transaction on the client. Once set, the client will not execute any commands until exec is called. Returns the Client instance, to make it easy to chain commands.

client.multi(); // Begins a multi-command transaction.

client.set('foo', 13); // Adds a set operation to the queue.

Parameters

None

Return value

  • Instance ⇐ Client
