decrby

key

decrement

Decrements an integer by a given decrement. If the key is not already in the database it will be created with the value of 0 minus the decrement provided. If you just want to decrement an integer by one using decr is easier.

client.incr('foo', function(initialVal) {
  assert(initialVal === 1);

  client.decrby('foo', 10, function(newVal) {
    assert(newVal === -9);
  });
});

Parameters

  • Key ⇐ string
  • Decrement ⇐ integer
  • Callback ⇐ function

Callback

  • NewValue ⇐ integer
